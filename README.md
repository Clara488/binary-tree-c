# Binary Tree in C — Estruturas de Dados

Implementação completa de uma **árvore binária em C**, contendo múltiplas funções.
Este projeto foi desenvolvido como parte dos meus estudos em **Algoritmos Estruturas de Dados**, com foco em lógica, ponteiros, alocação dinâmica e recursão.

---

# Estrutura do Projeto

src/
├── arvore.c        # Implementação das funções da árvore
└── TADarvore.h     # Arquivo de cabeçalho com os protótipos

examples/
└── main.c          # Exemplo de uso da árvore

input/
└── entrada.txt     # Arquivo de entrada com valores para montar a árvore
